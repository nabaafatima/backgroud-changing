# 🎨 Background Changing Project

A simple and interactive web application that dynamically changes the background color of the page with the click of a button.

🔗 **[Live Demo]()**

---

## 🚀 Features
* Random hex color generation.
* Real-time background color updates.
* Displays the current active hex code on the screen.
* Clean and minimal user interface.

---

## 🛠️ Tech Stack
* **HTML5** - Page structure
* **CSS3** - Styling and layout
* **JavaScript (ES6)** - Color generation logic and DOM manipulation

---

## 💻 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate into the project folder:
   ```bash
   cd backgroud-changing
   ```
3. Open `index.html` in your favorite web browser.
